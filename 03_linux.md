# Linux Commands

## System

```shell
lsb_release -a
```

```shell
systemctl status nginx
```

```shell
systemctl start nginx
```

```shell
systemctl stop nginx
```

```shell
systemctl restart  nginx
```

## User

```shell
sudo -i
```

```shell
whoami
```

## Kill a process

```shell
ps aux
```

```shell
htop
```

```shell
kill <pid>
```
