# Runtimes

## Install Go

https://go.dev/dl

```shell
curl -OL https://go.dev/dl/go1.21.4.linux-amd64.tar.gz
```

```shell
tar -C /usr/local -xvf go1.21.4.linux-amd64.tar.gz
```

```shell
echo 'export PATH=$PATH:/usr/local/go/bin' >> ~/.bashrc
```
